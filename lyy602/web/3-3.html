烧鹅

起源

烧鹅是广东菜中的经典之一，尤其是在香港和广州等地非常受欢迎。它采用了独特的烤制技术和丰富的调味料，味道鲜美、色泽诱人。烧鹅的历史可以追溯到宋代，经过千百年的演变，成为了如今广受欢迎的美食。

烧鹅最初起源于广东，尤其在香港的传统烧腊店中经常见到。它与另一道经典菜肴“烧鸭”相似，但烤制的方法和味道上有所不同。烧鹅的特点是外皮香脆，内里肉质嫩滑，油而不腻，色泽金黄，往往搭配特制的甜酱或酱油食用。

特点

烧鹅的最大特点就是其色香味俱佳，外皮经过烤制后，呈现出金黄色的光泽，且油亮酥脆，肉质鲜嫩多汁。通常，在烤制过程中，鹅身会涂上一层特别调制的糖浆，使其表面更加香脆，同时保留了鹅肉的原汁原味。吃上一口，肉质嫩滑，带有微微的甜香，外皮则脆而不油腻，口感丰富。

做法

制作烧鹅的过程需要精心准备，以下是烧鹅的经典做法：

  1. 准备食材：选择新鲜的鹅，通常使用的是肥美的肉鹅，鹅的体型要适中。将鹅去除内脏，清洗干净。
  2. 腌制：将清洗干净的鹅放入调味料中腌制。腌制的调味料通常包括生抽、老抽、糖、五香粉、八角、桂皮、蒜蓉、姜片等，腌制时间通常为12小时至24小时，使得味道渗透入鹅肉。
  3. 涂糖浆：在腌制后的鹅表面涂上一层糖浆，糖浆一般由蜜糖与水调配而成，涂抹均匀。这层糖浆可以让烤出的鹅皮呈现金黄色，外脆内嫩。
  4. 烤制：将腌制和涂好糖浆的鹅挂入烤炉中，采用炭火烤制的方法。烤制过程中，需要不断调整火候和翻动鹅身，确保烤制均匀。一般需要烤制约1-2小时，直到鹅皮呈现出光泽且酥脆。
  5. 切割：烧鹅烤好后，切割成适合食用的块状或片状。通常烧鹅的皮肉分明，外皮切成薄片，鹅肉切成小块。
  6. 配料：烧鹅通常搭配特制的酱汁，如甜酱、酱油或是蒜泥等，增添风味。食用时，常常配上一碗白饭或与其他烧腊一起享用。

烧鹅的制作技巧非常关键，尤其是烤制过程中火候的掌握。若烤制过度，鹅肉可能会过于干硬；而烤制不足，则可能影响口感的酥脆和肉质的嫩滑。

如何享用烧鹅

烧鹅通常以整只或者切块的方式上桌，适合与家人和朋友一起分享。在传统的广东餐桌上，烧鹅是宴席中的重要菜肴之一。食用时，可以搭配米饭、炒菜或是凉菜等，增添餐桌的丰富度。

烧鹅也常作为广东茶楼的招牌菜之一，人们在享受早茶时，常常配上一些烧鹅或者烧腊拼盘。鹅肉酥脆的外皮与嫩滑的肉质，搭配上一壶功夫茶，味道更是让人陶醉。
